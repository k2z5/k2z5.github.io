СМАРТ.

  • С- каждый новый поступающий файл кладется в нужное место
  • М- по чеклисту на чистоту
  • А- все файлы хранятся по чеклисту на чистоту, затрачивая на это 2 часа в месяц
  • Р- все новые файлы хранятся по чеклисту на чистоту, затрачивая на это 2 часа в месяц, для развития новых проектов
  • Т- до 5 января, все новые файлы хранятся по чеклисту на чистоту, затрачивая на это 2 часа в месяц, для развития новых проектов
